# Chart Visual Indicator

## Overview
- I use lot of Indicator for my charts and it's difficult to manage all of them.
- Also this Indicators are easy to delete by mistake and I have to setup them again.
- So I have created this indicator to manage all my chart visuals in one place with one click.
- This indicator will automatically setup all chart visual as per my preferred settings so I don't have to worry about them anymore. 

## Features
- All Moving averages 10/20/50/100/200 with my preferred settings.
- All time High Line
- Bull Snort Candle Markers
- Inside Bar Candle Markers
- Intraday Box to differentiate between different days
- 10 Day EMA Line (1D) for intraday charts
