- Main problem with trading view ath screener is that there's not way to find stocks which are within 10-15% below ATH
- I can import ATH stocks from my custom screener and use Screener Filter on top of that
- Basic Filter

    - Exchange : NSE
    - Market Cap : 5B(500 crore) to 5T(5,00,000 crore)
    - Price : 40rs to 6000rs
- Additional depending on requirement
    - Sector filter  
    - Industry filter
